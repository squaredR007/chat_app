# پیام رسان تحت وب 
در این پروژه یک پیام رسان تحت وب با قابلیت های متعددی پیاده سازی شده است که به شما این امکان را میدهد تا بتوانید پیام های متنی یا حتی مدیا های مختلف را برای دیگر کاربران ارسال کنید
# قابلیت ها 
1.ثبت نام یک یوزر جدید و لاگین با مشخصات آن یوزر
2.ایجاد چت خصوصی
3.ایجاد گروه 
4.امکان ارسال عکس و فایل علاوه بر پیام های متنی
5.امکان آرشیو و پین کردن پیام ها 
6.امکان سرچ بین چت ها و محتواهای متنی یک چت خاص 
7.تغییر اطلاعات کاربری نظیر یوزرنیم ، بیوگرافی و عکس پروفایل
8.تغییر تم

# پایگاه داده یا DataBase
در این پروژه از فایل های متنی (TXT) به عنوان دیتابیس استفاده شده است که پیاده سازی آنها در فاز 2 پروژه انجام خواهد شد . در حال حاضر اطلاعات کاربران در با استفاده از انواع خاص ArrayList و HashMap که thread-safe هستند ذخیره سازی میشود
